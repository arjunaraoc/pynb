Jupyter Python notebooks  for analysis of Archive.org collection view trends

Usage: Download and open in Juypter notebook and run.

Files:

arxtelcat.ipynb : Creates a catalog file with all ids in a collection and a language

totviewstrend.ipynb : Takes a random sample of ids from the catalog file and downloads views data, sums of a monthly basis and presents the trend chart.

Requirements:
Jupyter
Python 3
Pandas
Numpy
Plot.ly (account required for graphs to be saved)
